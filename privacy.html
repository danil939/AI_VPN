Политика конфиденциальности AI VPN

Дата вступления в силу: 2024-05-29

Ваше расширение AI VPN (далее — "Расширение") уважает вашу конфиденциальность и стремится защищать ваши персональные данные. В данной политике описано, какие данные мы собираем, как их используем и как обеспечиваем их защиту.

Какие данные мы собираем

  • IP-адрес пользователя — используется для предоставления VPN-сервиса и отображения местоположения.
  • Данные Telegram — Telegram ID, имя, username — используются для авторизации и поддержки.
  • Уникальный идентификатор пользователя (UID) — для учёта подписки и работы сервиса.
  • Данные о подписке — тип подписки, дата окончания.
  • Токен авторизации — для проверки статуса пользователя.
  • Время подключения — для отображения времени сессии.
  • Данные о платежах — только для обработки оплаты (через платёжные сервисы).

Как используются данные

  • Для предоставления и улучшения работы VPN-сервиса.
  • Для авторизации и поддержки пользователей.
  • Для обработки платежей и управления подпиской.
  • Для внутренней диагностики и устранения ошибок.

Кому передаются данные

  • Только вашему серверу — для авторизации, поддержки, оплаты.
  • Третьим лицам не передаются, за исключением случаев, предусмотренных законом или необходимых для обработки платежей (например, YooKassa).
  • Данные не продаются и не используются для рекламы.

Хранение данных

  • Данные хранятся локально в браузере пользователя (chrome.storage.local) и на сервере AI VPN.
  • Данные о платежах хранятся только в платёжных сервисах и не сохраняются на сервере AI VPN.

Как пользователь может управлять своими данными

  • Пользователь может запросить удаление своих данных, написав на support@aivpn.team.
  • Пользователь может удалить расширение в любой момент, что приведёт к удалению локальных данных.

Согласие пользователя

Используя расширение AI VPN, вы соглашаетесь с данной политикой конфиденциальности.

Контакты

По вопросам конфиденциальности и удаления данных обращайтесь: support@aivpn.team
